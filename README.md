#  Weather App

A React weather application that displays current conditions, a 3-day forecast, and the past 3 days of weather history for any city.

##  Features
- **Current weather** — temperature, feels like, wind speed, and humidity.
- **3-day forecast** — daily high/low temperatures and conditions.
- **3-day history** — weather data for the previous three days.
- **City search** — look up any city by name.
- **Detail views** — expanded hourly breakdowns for both forecast and history.

##  Tech Stack
- **Framework:** React 19 (with the React Compiler enabled)
- **Build Tool:** Vite 7 for bundling and dev server
- **API:** WeatherAPI.com for weather data
- **Icons:** `react-icons` for UI icons
